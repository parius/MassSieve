Rules of Parsimony

Since a given peptide sequence can map to more than one protein, a method is desired to reduce the complexity of the resulting set of proteins. The following parsimony rules and definitions are designed to reduce the number of proteins and yet still account for all of the observed spectra and isoforms.
Peptides
Indeterminate For a given scan, only the top scoring hit is used. If there is more than one match that ties for the top score, then the peptide is indeterminate.
Distinct A peptide that is assigned to exactly one protein.
Shared A peptide that is assigned to more than one protein.
Proteins
Discrete A protein identification that is identified by only distinct peptide(s).
Differentiable A protein identification that has at least one shared peptide, but can be distinguished from other proteins because it has at least one distinct peptide.
Superset A protein identification containing only shared peptides from other subset proteins where each subset is a proper subset.
Subset A protein identification contains peptides common to a larger set of peptides corresponding to another protein identification which is a superset.
Subsumable A protein identification contains shared peptides that can be distributed as subsets of two or more other proteins. Formally, subsumable proteins are simply another class of subsets.
Equivalent Protein identifications that are based on the same set of shared peptide(s).
